## Description

Quick coming soon is only for those who need to put a simple note on the site immediately while upgrade or maintenance.

## Main Features

- Lightweight
- Easy installation.
- Can put simple note
- Responsive
- Background image change
- Social Button (Facebook,Twitter,Youtube)
